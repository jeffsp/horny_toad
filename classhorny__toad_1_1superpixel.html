Detailed Description

A superpixel is a big pixel.

A superpixel describes a square pixel region in an image.

The row and column specify the top left corner of the superpixel in a raster of image pixels. The size specifies the length in pixels of one side of the superpixel.

The minimum superpixel size is 1.

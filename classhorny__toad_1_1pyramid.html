Detailed Description

template<typename T, class M = jack_rabbit::raster<T>, class Cont = std::vector<M>>
class horny_toad::pyramid< T, M, Cont >

pyramid container adapter

Implements a multi-resolution pyramid. The base of the pyramid contains a high resolution image. Higher levels contain lower resolution images.

Examples:
pyramid_example0.cc, and pyramid_example1.cc.

Detailed Description

template<typename T>
class horny_toad::oriented_sin< T >

An oriented sine wave function object.

This functor generates a sine wave function value given x, y coordinates. The idea is to use it to fill a matrix-like container with a sine wave grating using an algorithm that calls this functor for each element location.

Note that the matrix that it fills assumes that element 0, 0 is in the logical top left of the matrix.
